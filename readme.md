# Trieur de dossier

Ce script python prend en argument un dossier et va copier les fichiers trouvés dans tous les sous-dossiers vers un dossier (année/mois).

Cela permet en cas de foutoir dans un gros dossier de retrouver assez facilement ses fichiers (image notamment)

```bash
python main.py C:\Users\me\Pictures

```
